## 模型校准（谁的哪类判断更可信）

- **DeepSeek**（盘口/防反读数最准）：**反盘口、强队难大胜、平局、门将开挂、防反**这几类判断反复命中——卡塔尔受让赢盘、西0:0/沙特1:1(门将+难净胜)全验证。**冲突时明显偏 DeepSeek 的"强队会不会被偷/被逼平"盘感。** 短板：①遇真碾压场偏保守（德国7:1 押走盘空、挪威 1:4 押小胜空，第3次）——别让它判断"该不该上让球大胜"；②纯"谁赢"方向也不高（伊朗2-0穿盘落空）。**穿盘判断是"点位特异"的**：指名某侧"血崩点"(法国迪亚塔)就敢穿且准，泛泛"难净胜/便秘"时才偏保守。**区分**：它点名某门将崩/某边路错位 → 信它穿盘；只笼统说"便秘" → 当避雷、别跟它压小胜/受让。
- **Claude**（两极分化）：真碾压场敢顺势押大胜且对（德国7:1 押5-0、挪威穿盘押对，与 GLM 唯二/同阵营押对大胜）；但**第1轮强主"小胜"系统性高估**——西/比/沙特/伊朗押胜 0/4 全平。→ 战术细节丰富可信，但"强主小让球大胜/穿盘"打折扣，碾压场(制空+替补+刷净胜意图三齐)才信它的大胜。
- **GLM**（约 8/12 较准）：和 Claude 类似敢追碾压大胜（德国、挪威押对），正文判断常最透（澳土场看对）。但与盘口同质化高、有时"看对押错"（澳土押平）。
- **GPT / Kimi**：偏保守稳健，强主小胜场押对、碾压让球大胜场空；Kimi 常正文看对但落注太怂。
- **Gemini**（最不稳定/弱反指标）：batch1 六场 0/6 系统性逆向押冷全错，batch2 回到 3/6。其"爆冷叙事"噪声大，**仅作弱反向参考，别跟它博冷**。

## 嘉豪正文里重点看哪里（被验证可信的硬信号）

- **门将两类信号要分清**：①对方门将"摘高球/扑救评分低" + 己方三高点定位球 → **大胜/过盘**（瑞典5:1、苏格兰客胜、德国7:1；法国门迪3分→3:1、挪威方门将1分→1:4 双验证）；②对方门将"扑救评分高/开挂" → **低比分/平**（西0:0 马尔西奥、沙特1:1 奥韦斯）。"低评分门将 + 边路错位"双信号齐＝强队穿盘——这是区分"碾压"还是"闷平"的第一信号。
- **真碾压三件套**：强主"刷净胜球明牌意图" + 绝对制空碾压 + 替补厚度 → 顺势上**让球大胜**，别被"锋线吐饼数据"劝退（德国7:1 验证）。三件套不全 = 别追让球。
- **体能/休整差**（一方仅 7~9 天 vs 对方 10天+）→ 下半场塌方（厄瓜多尔7.1天、突尼斯8.5天 → 主胜）。但注意：新西兰仅9天却 2:2 逼平伊朗——休整差是倾向不是铁律。
- **强主"便秘苦战/被逼平风险/难净胜2球"措辞要当真**（比1:1、伊朗2:2、西0:0 全应验）。
- **红牌剧本是过度演绎**：嘉豪爱写"对位边卫犯规10红牌10必染红"，基本没真红（厄全程11人），只取"边路被压制"即可，别买"对方染红"衍生玩法。

## 买法倾向（历史教训）

- **第1轮（揭幕轮）"两家都看好的强队胜"打折扣**：西/比/沙特/伊朗四场强队共识**全部没赢**（全平/0:0）。第1轮平局/冷门高发，**别堆顺风强队串**——要么配平局/受让对冲，要么只买能单关的分散下。**但第2轮起强队开始兑现**（法/阿/奥三碾压），顺风强队胜/真碾压让-1 安全度回升，参考性减弱。
- **强主让球大胜（让-1/让-2）两类要分**：①真碾压三件套齐 → 敢上（德国、第2轮法/阿/奥 让-1 全穿盘）；②否则屡空（西让-2、比让-1、伊朗让-1、卡塔尔让2 全空）。默认对"让2及以上 + 对方有1个定位球准星(阿菲夫式)"反买受让/走盘。**但碾压场别跟 DeepSeek 的保守压受让/走盘**（挪威受让+2 顺 DeepSeek 便秘判断、被 1:4 打脸）。
- **强主小让球(让1)遇密防+神门将+己方浪射 → 易平/受让**（巴西1:1、加拿大1:1、比1:1）。这类比分平/受让平价值高于追主胜。
- **AI 6家一致 ≠ 安全**：加波1:1、巴摩1:1、荷日2:2 都是 6 家一致却全错；尤其当多家"最易打脸"段都指向同一平/冷剧本，**该剧本当主线而非尾部风险**。
- **净胜球普遍被高估**：小胜场实际常只赢1球（海地、科特迪瓦），**胜平负比精确比分稳**；大胜场弱队也可能偷1个（奥地利 3:1 非零封）→ 博比分优先"1球小胜"或只押胜、别押精确大比分。
- **方向对≠押法对**：看对球还要落到正确盘口选项（澳土场 GLM/Kimi 看对押平=白看）。
- **逆共识博冷（跟 DeepSeek 押强队会平/被偷）在第1轮性价比高**：佛得角+2@3.85 + 沙特平@4.3 ≈16.6倍两腿全中。可作激进档主轴，而非顺风串的附属。
- **真碾压场押半全场要押"主/主"、别押"平/X"**：碾压队往往上半场就领先、HT 不会是平——"挪威 半全场平/胜"飞(挪威半场已领先)，而"阿根廷 半全场主/主"中。用半全场替代未开单关的强队胜时押"主/主"。**注：半全场/半场玩法脚本判不了(数据源只有全场比分)，复盘时必须问用户中没中、别猜**(见 SKILL.md Track 2)。
